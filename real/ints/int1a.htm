INT 1Ah: Функции таймера.

AHФункция
0 Читать кванты времени
Выход: CX,DX - кол-во квантов времени (CX-старшая часть значения)
AL - было переполнение таймера
1 Установить кванты времени
Вход: CX,DX - кол-во квантов времени
2 Читать часы
Только AT-совместимые компьютеры
Выход: CH - Часы в коде BCD (1230 = 12:30)
CL - Минуты в коде BCD
DH - Секунды в коде BCD
флаг CF при отключённых часах
3 Установить часы
Только AT-совместимые компьютеры
Вход: CH - Часы в коде BCD (1230h = 12:30)
CL - Минуты в коде BCD
DH - Секунды в коде BCD
DL = 1 для опции "световые дни"
4 Читать дату
Только AT-совместимые компьютеры
Выход: CH - Столетие в коде BCD (1230h = 1230)
CL - Год в коде BCD
DH - Месяц в коде BCD
DL - Месяц в коде BCD
флаг CF при отключённых часах
5 Установить дату
Только AT-совместимые компьютеры
Вход: CH - Столетие в коде BCD (1230h = 1230)
CL - Год в коде BCD
DH - Месяц в коде BCD
DL - Месяц в коде BCD
6Установить сигнал
Только AT-совместимые компьютеры
В указаное время генерирует прерывание int 4Ah

Вход: CH - Часы в коде BCD (1230h = 12:30)
CL - Минуты в коде BCD
DH - Секунды в коде BCD
флаг CF при отключённых часах
7 Сбросить сигнал
Только AT-совместимые компьютеры
