Прерывания DOS

INT 20h: Завершить программу.
INT 21h: Сервис DOS.
INT 22h: Адрес завершения.
INT 23h: Адрес Control-Break.
INT 24h: Адрес обработчика критических ошибок.
INT 25h/26h: Абсолютные чтение/запись диска.
INT 27h: Завершиться, но остаться резидентным.
INT 28h: Квант времени DOS.
INT 2Eh: Выполнить команду DOS.
INT 2Fh: Мультиплексное прерывание (спулинг печати).
INT 33h: Работа с мышью.
