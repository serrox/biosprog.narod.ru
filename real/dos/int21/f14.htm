Функция 14h: читать из файла через FCB.

Вход:
AH=14h
DS:DX = адрес структуры FCB открытого файла

Выход:
AL: 0 - чтение сделано успешно. 1 - файл закончился раньше, чем начались данные. 2 - переполнение адреса памяти. 3 - неожиданный конец файла (считана часть данных).

Примечание:
Адрес, куда пишутся данные устанавливается функцией 10h прерывания 21h

9-11

Расширение файла

В блоке 128 записей
