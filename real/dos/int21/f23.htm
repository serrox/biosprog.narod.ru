Функция 23h: Дать размер файла через FCB.

Вход:
AH=23h
DS:DX = адрес структуры FCB неоткрытого файла

Выход:
AL: 0 - файл найден, FFh - файл не найден

Примечание:
По адресу DS:DX находится структура FCB. В таблице приведён её вид:
БайтыСмысл
00 - текущий диск, 1 - Диск А, 2 - Диск В и т.д.
1-8Имя файла
Если имя короче 8-ми символов, то то оно дополняется пробелами справа
9-11Расширение файла
12-13Номер текущего блока данных
В блоке 128 записей
После отрытия файла = 0.
14-15Логический размер записи
После отрытия файла = 0.
Для этой функции устанавливает размер подсчёта длинны
16-19Размер файла
20-21Дата создания или изменения
22-31Резерв
32Номер текущей записи в текущем блоке
33-36Номер записи произвольного доступа
После выполнения этой функции возвращает значение, при умножении которого на размер подсчёта длинны файла получается длинна файла в байтах
