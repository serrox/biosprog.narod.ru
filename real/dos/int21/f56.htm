Главная - Реальный режим - Прерывания DOS - INT 21h: Сервис DOS. - Функция 56h: Переименовать/переместить файл.

Функция 56h: Переименовать/переместить файл.

Вход:
AH = 56h
DS:DX = путь к файлу в формате ASCIIZ
ES:DI = новый путь к файлу в формате ASCIIZ
CL - маска аттрубута

Выход:
если CF установлен AX = код ошибки

Примечание: перемещение производится в пределах одного диска.
