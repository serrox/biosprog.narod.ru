Главная - Реальный режим - Прерывания DOS - INT 21h: Сервис DOS. - Функция 40h: Писать в файл.

Функция 40h: Писать в файл.

Вход:
AH = 40h
BX = описатель файла.
DS:DX = адрес буфера с информацией для записи.
CX - Число записываемых байт
Выход:
если CF установлен AX = код ошибки иначе, AX = число реально записанных байт.
