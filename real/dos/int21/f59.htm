Функция 59h: Получить информацию об последней ошибке.


Вход:
AH = 59h
BX = 0000h

Выход:
AX - Расширенный код ошибки
BH - Класс ошибка
BL - Рекомендуемое действие
CH - Источник ошибки
ES:DI указатель, для тех кодов ошибки, для которых необходимо

Примечание:
CL, DX, SI, BP, и DS могут быть изменены
