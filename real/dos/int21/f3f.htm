Функция 3Fh: Читать файл.

Вход:
AH = 3Fh
DS:DX = адрес буфера для записи прочитанной информации
BX - описатель
CX - число считываемых байт

Выход:
если CF установлен AX = код ошибки, иначе AX = число реально считанных байт
