Главная - Реальный режим - Прерывания DOS - INT 21h: Сервис DOS. - Функция 3Ch: Создать файл.

Функция 3Ch: Создать файл.

Вход:
AH=3Ch
DS:DX = путь к файлу в формате ASCIIZ
CX - Атрибут файла

Выход:
если CF установлен AX = код ошибки, иначе AX описатель.

Примечание:
если файл уже существует, то вся хранящаяся в нём информация удаляется.

Стандартные описатели:
0 - STDIN (стандартный ввод)
1 - STDOUT (стандартный вывод)
2 - STDERR (стандартный вывод ошибок)
3 - AUX (последовательный порт)
4 - PRN (параллельный порт)

Аттрибуты:
бит 0 - только для чтения
бит 1 - скрытый
бит 2 - системный
бит 3 - метко тома/диска
бит 4 - Каталог
бит 5 - архивный
бит 6 - резерв
бит 7 - (в NetWare) файл с общим доступом (т.н. "расшаренный" файл)
бит 8-15 - резерв
